🧠 BIDII Strategy Engine — How It Works
BIDII ELITE PRO uses a 15-factor confluence engine computed on real OHLC candles from Deriv's live API (requires API token). Each factor contributes to a bullish/bearish score:

1. Trend Structure (EMA 9/21/50/200) — Full stack alignment required for max score.

2. Momentum (RSI 14 — Wilder smoothing) — Correct Wilder EMA method, same as TradingView/MT4.

3. MACD (proper EMA array method) — EMA12/26 from full history. Signal = EMA9 of MACD line.

4. Bollinger Band Position — Price near lower = BUY zone. Near upper = SELL zone.

5. Stochastic Oscillator — %K/%D cross in overbought/oversold zones.

6. ADX Trend Strength — Confirms trend validity; weak ADX warns of choppy conditions.

7. Volume Gate (OBV + CMF) — Both must agree for full score. Synthetic volume flagged neutral.

8. RSI Divergence (weighted prerequisite) — Regular divergence +5 and REQUIRED for counter-trend reversals. Hidden divergence +3.

9. Candlestick Patterns — Engulfing, pin bars, morning/evening stars, Marubozu, doji and more.

10. Support / Resistance Proximity 🔑 — Detects swing-high/low levels from last 60 bars. +3 if price is within 0.7×ATR of a confirmed level in signal direction. S/R confluence is one of the strongest quality signals.

11. EMA21 Slope Quality — EMAs must be actively trending (not just stacked). Flat EMA = no score.

12. RSI Momentum Direction — RSI must be moving toward the signal direction over the last 4 bars.

13. Fresh EMA9×21 Cross 🔑 — Bonus +2 when the EMA9/21 crossover happened within the last 8 candles. Early-trend entries have the best R:R.

14. Trading Session Filter (Forex) — London+NY overlap (13:00–16:00 UTC) = +2 confidence. Single session = +1. Off-session = caution warning. Not applied to synthetics/crypto.

15. Minimum Alignment Gate — Signal is blocked unless at least 4 separate factors agree on direction. This is the most important profitability upgrade — it eliminates marginal setups.

ATR Volatility Filter — Bottom 20th percentile ATR = market coiling, signal suppressed.

Candle Close Timing — Signals on candles <80% complete show a warning.

Signal Expiry — Signals older than 3 candle periods flagged as stale.

Demo Mode Block — Without a Deriv API token, the tool shows DEMO WARNING and blocks signals. If you have a token but live data fails temporarily, it shows a specific retry message (not a false DEMO warning).
